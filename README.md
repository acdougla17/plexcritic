This project is to create a Plex critic that will analyze your collection like a pretentious movie buff friend looking through your shelves.

#How To Test
- npm run dev
- http://localhost:3000/

#Phases of Project
1) Basic Skeleton (DONE)
2) Initial connection to Plex (DONE)
3) Fetch all items in a single library
4) Database Layer
    - Create schema
    - Create route
5) Create analytics module
    - Basic queries
        total films
        never watched %
        avg days to first watch
        added this year vs watched this year
        most collection appearances
        no collection appearances
        Most frequent actor/director/writer
6) Create critic to call queries and provide feedback
7) Simple React frontend
8) Dockerize
